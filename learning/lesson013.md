Клавиши "Е" и "Ы" нажимаются одновременно указательным пальцем. Для этого указательный палец поставьте между этими клавишами. Аналогично нажимаются сочетания клавиш "О" и "У".


```
| с | в | к | т |    | е | о | а | и |
| д | б | п | н |    | ы | у | я | ь |

        как                       ал
|   | # | # |   |         |   |   | # | # |
|   | # |   |   |         |   |   |   |   |

        он                        ил
|   | # |   |   |         |   |   |   | # |
|   | # |   | # |         | # |   |   |   |

```
