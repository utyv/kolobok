Начинаем изучать аккорды. Одновременное нажатие нескольких клавиш позволяет вводить некоторые буквы и часто используемые последовательности букв.

```
        р                        ю
|   |   |   |   |         | # |   |   |   |
|   | # |   | # |         |   |   | # |   |

        м                        э
|   |   | # | # |         |   | # |   |   |
|   |   |   |   |         |   |   | # |   |

```
